# Game-of-Life

## Overview
Implementation of Conway's "Game of Life" using OOP, parallel processing, and 2D graphics to create a simple application.

## Features
- Choose between sequential, std::threads, or OpenMP threads to calculate data for upcoming generation
- Simple 2D graphics to rapidly visualize generations

## Tools
- C++
- SFML (v. 2.6.1)
- OpenMP
