# Game-of-Life

## Overview
Implementation of Conway's "Game of Life" using command line arguments to change processing method. (Uses SFML 2.6.1 for graphics)

## Features
- Choose between sequential, std::threads, or OpenMP threads to calculate data for upcoming generation
- Simple 2D graphics to rapidly visualize generations

## Tools
- C++
- SFML (v. 2.6.1)
- OpenMP
